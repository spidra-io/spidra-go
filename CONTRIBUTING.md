# Contributing to spidra-go

## Getting started

```bash
git clone https://github.com/spidra-io/spidra-go.git
cd spidra-go
go build ./...
```

No external dependencies — the standard library is all you need.

## Making changes

1. Fork the repo and create a branch
2. Make your changes
3. Run `go build ./...` and `go vet ./...` to verify
4. Open a pull request against `main`

## Reporting issues

Open an issue at [github.com/spidra-io/spidra-go/issues](https://github.com/spidra-io/spidra-go/issues).
